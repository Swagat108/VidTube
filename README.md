#  VidTube Backend

> A production-grade YouTube-like video hosting backend built with Node.js, Express.js, and MongoDB.

---

##  About the Project

This is a **complete backend project** for a video hosting platform similar to YouTube. It is built following industry-standard practices and covers everything from authentication to user interactions.

**Tech Stack:**
- **Runtime:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB
- **Auth:** JWT (Access Tokens & Refresh Tokens)
- **Security:** bcrypt for password hashing

**Features include:**
-  User Signup & Login
-  Video Upload
-  Like & Dislike
-  Comments
-  Subscribe & Unsubscribe
-  Secure token-based authentication
